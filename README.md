### 五、项目介绍
项目编号：python080

全国民宿信息管理系统利用计算机网络实现信息化管理，使整个全国民宿信息管理的发展和服务水平有显著提升。
系统拥有管理员和用户、前台管理员三个角色，主要具备注册登录、个人信息修改、用户信息、前台管理员信息、房间信息、房间预订、客房退订、入住登记、退房办理、换房登记、服务请求等功能模块。将纸质管理有效实现为在线管理，极大提高工作效率。
